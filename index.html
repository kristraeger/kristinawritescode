  • My SciFi Name Generator

    Built when i first learnt about search algorithms! Channeling my inner Sci-Fi nerdette (or so). Because who doesn't wonder what your character name would be like in a SciFi World? When i didn't find a name generator which actually uses my first name as input, i just built one myself! And it has BB-8 to play around with! :)

    The name generator constructs and uses Markov chains. Given a list of sample names from Kate Monk's Onomastikon as well as all popular sci fi series, it analyzes each name to construct frequency tables for name length and letter pairs. It then uses these tables to construct a new name in the style of the sample set.

    The result is optimized to match as many characters of your name as possible, depending on the length of your name and the amount of unique characters in your name. If you don't like the first match - just keep klicking for more names!

    Mine is kriststar (duh). What is yours?!
